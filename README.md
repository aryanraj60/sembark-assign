# react-ts-vite

## Getting Started

### Prerequisites

1. Install node js `>= 16.13.0`
2. Install `npm` package manager for installation instructions.

### Quick Start

1. Clone the repo and go to the repository directory.
2. Install the packages.
   ```sh
   npm install
   ```
3. Start the app in development mode
   ```sh
   npm run dev
   ```

## Tech Stack

- React JS with Vite
- TypeScript
- Jest for testing
